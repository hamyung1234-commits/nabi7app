## 구현된 기능

### 1. 한국어 번역
모든 기존 8개 페이지의 영어 텍스트가 한국어로 변환됨:
- UI 라벨, 버튼, 헤더, 테이블 헤더
- 상태 메시지, 빈 상태 메시지
- Placeholder 텍스트

### 2. 고객정보 (Customer CRM) 카테고리
신규로 추가된 9번째 카테고리:

#### 기본 정보 입력 필드
- 고객명, 연락처 (클릭 시 전화 연결 표시)
- 고객 유형 (매도자 / 매수자 / 양방향)
- 담당자, 최초 거래일, 최근 거래일
- 관심 종목 (복수 입력 가능)
- 메모 (고객 특이사항, 성향, 주의사항)
- 계좌 정보 (은행명, 계좌번호, 예금주)
- 상태 (활성 / 휴면 / 블랙리스트)

#### 자동계산 기능
거래내역 연동:
- 총 거래 건수 (해당 고객 거래내역 행 수)
- 총 매수/매도 거래금액
- 총 수수료 수입
- 평균 거래금액
- 최초/최근 거래일 자동 추출
- 거래 공백일수 (30일+ 경고, 90일+ 위험 표시)

의뢰내역 연동:
- 총 의뢰 건수
- 진행중/완료 의뢰 수
- 의뢰 완료율

#### 화면 구성
- 상단 요약 카드: 전체 고객 수 / 신규 고객 / 활성 / 장기 미거래
- 고객 카드형 목록 (이름, 연락처, 최근 거래일, 상태 배지)
- 검색: 이름, 연락처, 관심종목
- 필터: 고객 유형 / 상태
- 정렬: 최근 거래일 / 거래 금액 / 수수료 수입

#### 고객 상세 페이지
- 탭 구성: 기본 정보 / 거래내역 / 의뢰내역 / 메모
- 거래 공백일수에 따른 색상 경고 (노랑/빨강)
- 계좌번호 마스킹 표시 (클릭 시 전체 표시)

#### 엑셀 내보내기
- 헤더 메뉴의 엑셀 내보내기에 고객정보 시트 포함
- 고객 CRM 페이지에서 별도 엑셀 내보내기 버튼 제공
