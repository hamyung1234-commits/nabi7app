## URL 전달 시 주의사항

다른 사람에게 URL을 공유할 때는:

1. **전체 복사**: `https://` 포함하여 전체 선택
2. **붙여넣기**: 메시지나 메모장에 붙여넣기 후 전송
3. **접속 확인**: 받는 사람이 직접 클릭해서 접속 확인

## 그래도 안 열린다면: 새로운 URL 생성

### 방법 1: Netlify Drop (가장 빠름 - 1분)

1. 아래 링크로 이동:
   👉 **https://app.netlify.com/drop**

2. GitHub 계정으로 로그인 (이미 브라우저에 로그인되어 있음)

3. **dist** 폴더를 드래그 앤 드롭
   - 폴더 경로: 이 프로젝트의 `dist` 폴더

4. 자동 생성된 URL로 바로 사용 가능!
   - 예: `random-name-12345.netlify.app`

---

### 방법 2: GitHub 저장소 이름 변경

저장소 이름 `-nabi-app-` → `nabi-app`으로 변경:

1. https://github.com/hamyung1234-commits/-nabi-app-/settings 접속

2. **Repository name** 필드에서:
   - `-nabi-app-` → `nabi-app` (앞뒤 하이픈 제거)

3. **Rename** 클릭

4. 새 URL: **https://hamyung1234-commits.github.io/nabi-app/**
