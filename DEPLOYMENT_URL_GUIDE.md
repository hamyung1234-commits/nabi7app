### 저장소 이름 변경 절차

#### 1단계: 현재 저장소 이름 변경
1. GitHub에 로그인: https://github.com/hamyung1234-commits/-nabi-app-/
2. **Settings** (설정) 클릭
3. **Repository name** 필드에서 `-nabi-app-` → `nabi-app` 로 변경
4. **Rename** 버튼 클릭

#### 2단계: GitHub Pages 재설정
1. 변경된 저장소로 이동: https://github.com/hamyung1234-commits/nabi-app/
2. **Settings** → **Pages** 이동
3. **Source**: "Deploy from a branch" 선택
4. **Branch**: `master` / `/(root)` 선택
5. **Save** 클릭

#### 3단계: 새 URL 확인 (2-3분 대기)

## ❓ 자주 묻는 질문

**Q: 다른 사람에게 URL이 안 열린다고 합니다.**
A: GitHub 저장소가 Public으로 설정되어 있는지 확인하세요.
   - Settings → Danger Zone → Make public 클릭

**Q: URL을 어떻게 복사하나요?**
A: 위 주소를 더블 클릭하여 전체 선택 후 Ctrl+C (또는 우클릭 → 복사)

**Q: 새 탭에서 열리나요?**
A: 네, 브라우저 주소창에 붙여넣기 후 Enter를 누르세요.
