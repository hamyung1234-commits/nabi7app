# 🚀 Netlify 즉시 배포 가이드

## 방법: 드래그 앤 드롭 (가장 간단!)

### 1단계: dist 폴더 열기
파일 탐색기에서 이 프로젝트의 `dist` 폴더를 찾으세요.

### 2단계: Netlify 드롭 접속
브라우저에서 다음 주소로 이동하세요:
```
https://app.netlify.com/drop
```

### 3단계: 배포
1. `dist` 폴더를 Netlify 페이지의 점선 영역에 드래그하세요
2. 잠시 기다리면 새 URL이 생성됩니다!

## 생성될 URL 형식
```
https://-random-Name-12345.netlify.app
```
